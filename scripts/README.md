# Scripts Layout

- `build/`: Docker image build entrypoints.
- `prep/`: dataset preparation and export scripts.
- `run/`: main partitioning pipeline entrypoints.
- `query/`: Neo4j query benchmarking scripts.
- `materialize/`: partitioned property-graph materialization.
- `results/`: result merging, summarization, and chart generation.
- `wrappers/`: Scotch/PT-Scotch compatibility wrappers.
